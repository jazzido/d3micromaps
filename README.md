d3.micromaps
============

A prototype implementation of [Linked Micromap Plots](http://mason.gmu.edu/~dcarr/lib/v9n1.pdf) (Daniel B. Carr et al. 1998), applied to some variables of the 2010 Argentinian Population Census.
